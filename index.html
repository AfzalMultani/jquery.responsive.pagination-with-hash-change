Usage as in this Example

Feel free to use CSS Ids ,-)

HTML


<button class="prev">prev</button>
<button class="next">next</button>
<span class="counter"> </span>
<div class="paging-nav"></div>
<ul id="gallery">
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>


    <li>40</li>
</ul>
<button class="prev">prev</button>
<button class="next">next</button>
<div class="paging-nav"></div>

jQuery

Use the plugin as follows:


$('#gallery').ResponsivePagination({
   nextSelector: '.next',
   prevSelector: '.prev',
   counterSelector: '.counter',
   pagingSelector: '.paging-nav',
   itemsPerPage: 5,
   initialPage: 1,
   showPageNav: true,
   showPageNumber: true,
   textCounter: 'Page: ' 

});

HTML/CSS

The generated HTML source in <div class="paging-nav" > </div>: 


<ul>
    <li class="active"><a href="#1">1</a></li>
    <li><a href="#2">2</a></li>
    <li><a href="#3">3</a></li>
    <li><a href="#4">4</a></li>
    <li><a href="#5">5</a></li>

    <li><a href="#30">30</a></li>
</ul>

Notes

  • Requires jQuery 1.11+.
  • the # hash of a linked URI is used
  • tested in IE 10, IE 11, Firefox, Chrome
